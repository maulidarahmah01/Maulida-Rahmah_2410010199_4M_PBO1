## Deskripsi

SecondWear merupakan aplikasi berbasis Java yang digunakan untuk mengelola penjualan baju preloved.

Program memiliki fitur:

- Login Admin
- Tambah Barang
- Lihat Barang
- Transaksi Penjualan
- Laporan Penjualan
